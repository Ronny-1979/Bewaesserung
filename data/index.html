🔌 Sensoren aktivieren/deaktivieren

Deaktivierte Sensoren werden von der Pumpensteuerung komplett ignoriert. Regen-Sensor deaktiviert → wirkt wie „kein Regen". Wasserstandssensor deaktiviert → wirkt wie „Wasser vorhanden". Praktisch z.B. wenn ein Sensor defekt ist oder gerade nicht angeschlossen ist.

ℹ Wie funktioniert das?
Der ESP32 aktiviert automatisch den passenden internen Widerstand — kein externer Widerstand nötig.

HIGH = aktiv gewählt:
→ Interner PULLDOWN aktiv → Pin liegt ohne Sensor stabil auf LOW (= inaktiv)
→ Sensor schließt Pin nach 3,3V → HIGH → Sensor „aktiv"
→ Typisch wenn Kabel 2 deines Sensors an GND hängt

LOW = aktiv gewählt:
→ Interner PULLUP aktiv → Pin liegt ohne Sensor stabil auf HIGH (= inaktiv)
→ Sensor schließt Pin nach GND → LOW → Sensor „aktiv"
→ Typisch wenn Kabel 2 deines Sensors an GND hängt und schaltet
⚙️ Pumpen-Relais-Polarität

Stellt ein wie das Relais-Modul geschaltet wird. Viele günstige Module sind „active LOW" — die schalten verkehrt herum. Falls die Pumpe direkt beim Booten anläuft → andere Polarität wählen.

📡 Funk-Empfänger QIACHIP RX480E (fest im Code)

433 MHz, Momentary-Modus (1× Lerntaste). D0/D1 direkt an ESP32, kein Widerstand nötig.

ℹ Anlernen: Lerntaste 1× → LED erlischt → Taste A drücken → LED blinkt 3× → Lerntaste 1× → Taste B drücken → fertig.
🕐 Datum & Uhrzeit

Kein NTP/Internet. Die Uhr läuft intern weiter und wird alle 5 Minuten gesichert — nach einem Stromausfall wird die zuletzt bekannte Zeit automatisch wiederhergestellt (nur näherungsweise, siehe Hinweis unten). Für genaue Zeit trotzdem kurz prüfen/neu setzen.
